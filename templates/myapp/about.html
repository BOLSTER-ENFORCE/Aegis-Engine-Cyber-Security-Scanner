About Aegis Engine

Aegis Engine is a local cybersecurity scanner and EDR platform for learning, auditing, and reporting. It combines system, process, startup, network, file, threat-intelligence, MITRE ATT&CK, incident response, and reporting modules into a single Django workflow.

Architecture

🔍 Collection

Asset discovery, hardware fingerprinting, compliance checks, endpoint protection assessment, user audit trail, network exposure analysis.

🧠 Analysis

Process analysis, startup item audit, malware signature matching, file reputation, MITRE ATT&CK mapping, behavioral detection, risk scoring.

🚨 Response

Incident management, quarantine engine, automated remediation, forensic snapshots, kill chain analysis, threat timeline.

📄 Reporting

Executive PDF reports, JSON export, real-time dashboard, threat intelligence feed, compliance dashboard, risk metrics.

Technology Stack

Backend

Django, Python, psutil, watchdog, reportlab

Frontend

Bootstrap 5, Tailwind CSS, Chart.js, custom dark neon theme

Security

22 integrated security modules, real-time monitoring, IOC correlation, MITRE framework
